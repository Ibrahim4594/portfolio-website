# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2025-12-13

### Added
- Introduction video with mobile phone frame display
- Custom preview image for video thumbnail
- Video lazy loading for improved performance
- New UN SDG certificate (new32.jpg)

### Changed
- Optimized video loading from `preload="metadata"` to `preload="none"`
- Reduced initial page load time by ~10 seconds
- Updated video poster from profile-pic.jpg to preview.jpg

### Performance
- Video now only loads on user interaction
- Page load reduced from 77MB to 110KB initial load
- Improved Core Web Vitals scores

## [1.1.0] - 2025-12-12

### Added
- Comprehensive README with badges and documentation
- Professional footer with 3-column layout
- Enhanced CSS with cubic-bezier easing
- Improved button animations with gradient sweep
- Skills section with 4 categories
- 15 gallery sections with 40+ certificates
- 6 featured AI/ML projects

### Changed
- Updated navigation logo to IBRAHIM_SAMAD
- Polished visual design throughout
- Enhanced hacker-card styling with stronger borders
- Improved skill tags with scale on hover

### Fixed
- Contact form submission blocking issue
- EmailJS CORS errors (switched to FormSubmit.co)
- Project images opacity issue

## [1.0.0] - 2025-12-11

### Added
- Initial release of portfolio website
- Matrix rain background animation
- Terminal-style typography and theme
- Hero section with typing effect
- About section with achievements
- Projects section with custom-generated images
- Gallery section with video support
- Contact form with FormSubmit integration
- Responsive design for all devices
- Vercel deployment

### Features
- Next.js 15 with App Router and Turbopack
- TypeScript for type safety
- Tailwind CSS with custom cyberpunk theme
- Glass-morphism UI components
- Custom CSS animations (typing, glitch, hologram, neon pulse)
- Working contact form with email delivery
- Video support in gallery with hover-to-play

### Types of Changes
- **Added** for new features
- **Changed** for changes in existing functionality
- **Deprecated** for soon-to-be removed features
- **Removed** for now removed features
- **Fixed** for any bug fixes
- **Security** for vulnerability fixes
